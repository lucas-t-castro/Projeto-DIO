# Repositorio De Projeto DIO

Este é meu primeiro (de muitos) repositorio no github. 

Nele vou commitar meus futuros projetos pela plataforma DIO, onde a cada projeto evoluindo a cada projeto.



## Sobre o projeto

Fiz esse repositorio cpm bases nas aulas de Introdução ao GIT e GITHub do professor Otávio Reis. Criei seguindo os seguintes passos:

- Criei um repositorio no GITHub

- Copiei o link do repositorio

- Criei uma pasta na minha maquina Local

- Criei um documento README txt e coverti para md para usar no repositorio

- Usei os comandos ls, git status, git init, git add, git commit -m, git branch -M main, git push -u origin main para seguir e finalizar o projeto
